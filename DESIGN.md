# 设计系统 — Kagami 镜

> 方向代号：**晒褪了色的蒙德里安 / The Painted Ledger**
> 任何视觉 / UI 决策前先读本文件。未经明确批准不要偏离。
> 本设计系统由 `/design-consultation` 产出（2026-06-30），并刻意**不再使用 shadcn 的有样式组件与默认 slate 主题**——保留 Radix 无样式行为基元（管焦点 / 键盘可访问性），其余 class 体系与组件外观全部自定义。

## 产品语境

- **这是什么：** Kagami（镜）的 Web 管理台，是一扇观察「一个拥有自己生活的 Agent」内心活动的窗。不是聊天机器人 UI。
- **谁在用：** 仅创建者一人（私人观测台，不对外）。
- **品类 / 邻居：** 开发者观测台 / Agent 运行时仪表盘（精神近邻：Linear、Grafana 的克制，但刻意不抄它们的视觉语言）。
- **项目类型：** 数据密集的内部 Web 仪表盘。承载中文（CJK），到处是 ID、时间戳、token 计数、JSON。

## 美术方向

- **方向：** 新造型主义骨架 + 古典绘画颜料色（蒙德里安的结构 + 文艺复兴 / 印象派的色相与空气）。
- **统一法则（不可动摇）：**
  - **几何永远利落**——网格 / 分隔线 / 面板边界一律 `2px` 暖近黑硬线，永不模糊、永不圆角堆叠。
  - **颜色永远是颜料调出来的**——没有一个数字纯色（绝不 `#FF0000`），每块都是晒暗、微去饱和的颜料色；**色块内永不做渐变**。
  - 硬与软被分配到不同图层，所以它们不打架。
- **装饰层级：** intentional（克制）。靠暖中性纸感底色 + 硬线网格构图，不靠纹理 / 阴影 / blob。
- **心情：** 第一眼「这像一幅画，不像一个后台」——克制的敬意，然后才发现它是活的密集仪表。
- **配色铁律：** 颜色是**配给，不是涂抹**。90% 界面只有两种暖中性 + 暖近黑；饱和色只落在「此刻活着 / 有语义」的数据上。一块红出现，必须有意义。

## 字体

- **Display / 大数字（拉丁）：** Fraunces — 变量光学字号，文艺复兴刻字般的笔锋，承载「绘画」气质；用于页面标题、关键状态名、大统计数字，不大面积使用。
- **CJK 标题：** 思源宋体 Noto Serif SC — 宋体的庄重与老大师同源；**宋体做中文标题是本系统的标志性冒险**，让数 token 的工具读起来像 Agent 在写日记。
- **正文（拉丁）：** Literata — 纸本文字感，适合观测日志与长说明。
- **正文 / UI 标签（CJK 密集）：** 思源黑体 Noto Sans SC。
- **数据 / 表格 / 代码（必须 tabular-nums）：** JetBrains Mono — 承载所有 ID / token / 时间戳 / JSON / provider payload。数据是 Kagami 的「笔触」，不藏，用漂亮等宽呈现，让技术密度本身成为审美。
  - 备选升级（付费）：Commit Mono；拉丁正文亦可换 Söhne / 标题换 Cormorant Garamond / GT Sectra（同档审美）。
- **加载：** Google Fonts（或 Bunny Fonts）`<link>`；CJK 子集化按需自托管以控体积。
- **字号刻度（rem，base 16px）：** 12 / 13 / 15（正文）/ 17 / 20 / 24（区块标题）/ 34（中文 display）/ 52（拉丁 display）。

## 色彩

- **取向：** restrained-meaningful（少量色 + 大面积中性，色彩稀有且带语义）。
- **语义映射（一块色 = 一种含义）：**
  - 朱砂红 `#B23A2C` — 错误 · 主动事件 · Agent 发言
  - 群青蓝 `#284C82` — LLM · 推理 · context · provider
  - 赭黄 `#D7A12C` — scheduler · 等待 · pending
  - 绿土 `#5F7C5E` — Story · 记忆 · recall
  - 茜草玫红 `#A64B4B` — 高 token 成本 · 风险信号（少量）
- **中性（浅色 / 读档案）：** 底 `#F1EBDF`（石膏暖白）· 面 `#FAF6EE`（亚麻）· 凸面 `#EFE3D0` · 细线 `#D9CBB6` · 弱字 `#766C5E`（生赭）· 墨 `#1C1814`（骨黑，非纯黑）。
- **深色 / 守夜（夜间画室，明暗对照，不是反相）：** 底 `#15110C`（沥青棕黑）· 面 `#201B14` · 凸面 `#2A2218` · 细线 `#382E22` · 弱字 `#8E8170` · 墨 `#ECE4D2`（烛光铅白）。色块在黑底上被点亮：红 `#CF5340` · 蓝 `#5680C0` · 黄 `#E6B84A` · 绿 `#82A079` · 茜 `#C96A63`。
- **主用模式：** 浅色（`:root` 默认，对应当前前端）；深色作为 `.dark` 完整重设（非简单取反），随时可切。

## 间距

- **基础单位：** 4px。
- **密度：** comfortable（观测台偏紧，但留出画布般的外边距）。
- **刻度：** 2 / 4 / 8 / 12 / 16 / 24 / 32 / 48 / 64。

## 布局

- **取向：** grid-disciplined，但**非对称**——一块画板，不是等宽卡片流。**失衡是刻意的**：最大的「现在 / live context」面占左上偏心处，眼睛先到那；次要面板按权重缩放。
- **网格线：** `2px` 暖近黑硬线作为结构本身（蒙德里安线），可满铺。面板边界即构图的一部分。
- **侧边导航：** 一条窄的暖近黑「画布边」竖栏。
- **最大内容宽：** ~1180px。
- **圆角：** 结构层 **0**（硬朗）；重复列表行最多 `4px`；不做统一气泡圆角。

## 动效

- **取向：** minimal-functional——只保留帮助理解的过渡。颜料色块的出现 / 消失要果断，不做花哨编排。
- **缓动：** 入 ease-out / 出 ease-in / 位移 ease-in-out。
- **时长：** micro 50–100ms · short 150–250ms · medium 250–400ms。

## 实现备注（去 shadcn）

- 保留 `@radix-ui/*` 无样式基元（dialog / select 的焦点陷阱与键盘可访问性），**移除 shadcn 的有样式组件层与默认 slate 主题**。`class-variance-authority` 作为中性的 class 组合工具保留（button / badge 的 variant），不再承载 shadcn 默认观感。
- `apps/web/src/index.css` 的 `:root` / `.dark` CSS 变量已改为本文件的颜料盘（HSL 通道三元组，沿用 `hsl(var(--x))` + 透明度修饰符）；Tailwind 沿用，并新增 5 个语义颜料色 + 侧栏 token。
- 字体经 `index.html` 的 Google Fonts `<link>` 加载；`tailwind.config.ts` 配 `font-serif`（Fraunces / 思源宋体）、`font-sans`（Literata / 思源黑体）、`font-mono`（JetBrains Mono），结构边界默认 `border` 宽度提到 2px、`--radius` 归零。
- 组件外观（button / card / table / badge / dialog / select / chart / json-panel / mobile-card）与 layout（侧栏画布边竖栏、列表/详情画框）已按本系统重写；`AuthPage` 等自带 slate/粉彩样式的页面已并入颜料盘。
